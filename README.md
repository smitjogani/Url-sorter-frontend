# TinyLink Frontend

React frontend application built with Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. (Optional) Create `.env` file:
   ```env
   VITE_API_URL=http://localhost:5000
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- Dashboard with link management
- Create short links
- Search and sort links
- View individual link statistics
- Delete links
- Responsive design

## Tech Stack

- React 18
- React Router
- Vite
- CSS (no framework)
